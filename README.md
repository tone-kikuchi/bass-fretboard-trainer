# Bass Fretboard Trainer

ベースのスケール練習とターゲット練習をまとめた、GitHub Pages 向けのシングルページアプリです。

## 特徴

- 4弦/5弦切り替え + 24フレットの指板を固定表示
- キー/スケール/コード/進行プリセット選択と多彩なチューニングプリセット
- ノート名/ルート/度数/インターバル表示とスケール/コードトーン/ガイドトーンのレイヤー表示
- スケール再生（選択したキー/スケールを順番に再生）
- 4つの練習モード（ノート当て/度数当て/コードトーン着地/ガイドトーン）
- 進行ビューでテンポ再生とバーごとのガイドトーン可視化
- 練習統計を localStorage に保存
- 日本語/英語の切り替えに対応

## 開発

```bash
npm install
npm run dev
```

## デプロイ

`main` ブランチへの push で GitHub Pages に自動デプロイされます。

## 注意

このリポジトリはパブリック公開向けのため、セキュア情報やプライバシー情報は含めない構成になっています。
